TotalRecoilJS

Developer Tool that debugs your Recoil-React Applications

Why TotalRecoilJS Was Created

Recoil.js is a new experimental state management library recently opensourced by Facebook. However, we realized that there are currently no developer tools out there making it scary for developers to integrate Recoil!

We created a Chrome Extension for Recoil-React applications that will help developers visualize and debug state!

Installation

  • Install Chrome Extension (Currently go to our github and clone repo, run "npm install" and "npm run start")
  • Head over to React-Recoil Application
  • Right Click -> Inspect -> Select Recoil Panel
  • Trigger State Change to view component/selector tree data
